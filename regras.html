Regras gerais

  • O objetivo do jogo é acertar o número secreto.
  • Devido a API de reconhecimento de voz, esse projeto funcionará melhor no Google ou Edge.
  • Essa aplicação tem reconhecimento de voz, então, para você chutar um número, basta falar. Só lembre-se de pertmitir o acesso ao seu microfone.
  • A cada tentativa o sistema irá lhe apresentar uma dica, informando se o número secreto é maior ou menor do que o número que você chutou.
  • Quando você acertar, o sistema irá lhe informar o o acerto e qual o número correto e também lhe dará a opção de reiniciar o jogo com um novo número secreto
